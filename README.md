# AI-Powered Tax Assistant

## Setup Instructions
### 1. Backend (Spring Boot)
- Install Java & Maven
- Navigate to `backend/` and run: `mvn spring-boot:run`

### 2. Frontend (React.js)
- Navigate to `frontend/`
- Install dependencies: `npm install`
- Start React app: `npm start`

### 3. Machine Learning (OCR)
- Install dependencies: `pip install pytesseract opencv-python`
- Run OCR script: `python ml/ocr.py`

### 4. Database Setup (PostgreSQL)
- Run `schema.sql` in PostgreSQL to create the table
